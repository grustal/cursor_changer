 Cursor Changer

Create, customize, generate, and apply Windows mouse cursors with ease.

A modern desktop application that allows users to create custom cursors from images, generate AI-powered cursor designs, install cursor packs, and manage Windows cursor schemes through a simple and intuitive interface.

 ✨ Features

 🎨 Cursor Creation

* Create custom Windows cursors from images
* Convert PNG, JPG, JPEG, BMP, ICO, and other formats to `.cur`
* Automatic image resizing and optimization
* Hotspot configuration support
* Real-time cursor previews

 🤖 AI Cursor Generation

* Generate unique cursor designs using AI
* Create themed cursor packs from text prompts
* Customize generated cursors before installation

 📦 Cursor Pack Support

* Install complete cursor packs from ZIP archives
* Automatic extraction and organization
* Support for community-created cursor themes
* One-click theme installation

 🖱️ Cursor Management

* Apply cursors directly to Windows
* Automatic cursor-role detection
* Manage multiple cursor schemes
* Restore default Windows cursors
* Preview changes before applying

 🔧 Advanced Features

* Drag-and-drop file support
* RealWorld Cursor Editor (`.crs`) compatibility
* Cursor scaling and positioning
* Batch cursor processing
* Modern and user-friendly interface

 🚀 Installation

 Requirements

* Python 3.9+
* Windows 10/11


 Install Dependencies

```bash
pip install -r requirements.txt
```

 Run

```bash
python main.py
```
